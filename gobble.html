Project Gobble

"Gobble" is a dining review app that simplifies choosing restaurants by offering personalized recommendations.

Key Features

  • Review Aggregation: Combines reviews from Google Business, SmartLocal, TikTok, and user feedback.
  • Personalized Recommendations: Suggests eateries based on user profiles, preferences, and budgets.
