Project REscue

A comprehensive disaster relief platform offering real-time assistance and emergency updates.

Key Features

  • Emergency Assistance: Request help instantly with real-time notifications to responders.
  • Live Disaster Updates: Track affected areas, severity levels, and safety zones.
  • Help Station Locator: Find the nearest aid centers with resource availability.
  • Admin Dashboard: Manage and track relief efforts efficiently.
